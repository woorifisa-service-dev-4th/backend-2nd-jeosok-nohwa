# 💪저속노화, jeosok-nohwa💪


<br>

## 🛠 프로젝트 개요

이 프로젝트는 **MySQL DB와 연결**하여 **사용자 관리, 1:1 채팅, 뉴스 크롤링 및 관리 기능**을 제공합니다.

<br>

## 📌 주요 기능

1.사용자(로그인,회원가입)
- 사용자 데이터 저장 및 관리
- log4J 로깅 사용

2.채팅
- 소켓으로 1대1 채팅
- 추후 mongoDB로 채팅 저장 예정

3.뉴스 및 소식

- 네이버 뉴스 크롤링
- 뉴스 데이터 저장 및 관리
  - ✅ 뉴스 생성
  - ✅ 뉴스 조회
  - ✅ 뉴스 수정
  - ✅ 뉴스 삭제
